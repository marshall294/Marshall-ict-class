Year 1 ICT — Week 5

Topic: Using the Keyboard to Edit Text | Teacher: Marshall Jacob

Learning Objective

To learn how to use the keyboard to make changes to text on the computer.

Success Criteria

  • ✔️ I can use the keyboard to type and change letters or words.
  • ✔️ I can use the Backspace or Delete key to remove unwanted letters.
  • ✔️ I can use the Spacebar, Enter, and Shift keys correctly.

Notes & Explanations

The keyboard helps us to write words, numbers, and symbols on the computer. We can also use it to edit text — that means to change or correct what we have typed.

  • Backspace Key: Removes letters from the left side of the cursor.
  • Delete Key: Removes letters from the right side of the cursor.
  • Spacebar: Adds a space between words.
  • Enter Key: Moves the cursor to a new line.
  • Shift Key: Helps you type capital letters and symbols.

Example:

If you type “Ths is fun”, you can use the Backspace key to remove the letter “s” and type “i” to make it correct: “This is fun.”

Editing text helps to make your work neat and correct.

Key Concepts

  • Edit: To change or correct what you have written.
  • Cursor: The blinking line that shows where typing will appear.
  • Backspace/Delete: Keys used to remove text.
  • Shift Key: Used to make capital letters or type symbols.
