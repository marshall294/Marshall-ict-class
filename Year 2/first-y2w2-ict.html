Learning Objective

By the end of this lesson you will be able to identify the uses of Information Technology (IT) in your school and sort devices by the jobs they do.

Success Criteria (I can…)

  • identify examples of IT in school (computers, projectors, printers, tablets, Wi-Fi, etc.)
  • explain why some IT devices can be used for more than one purpose
  • sort school IT into groups depending on what they are used for

What is Information Technology (IT)? — Full explanation

Information Technology (IT) means using computers, devices and systems to store, find, change and share information. IT is everything that helps people work with information faster and better.

Key parts of IT (in simple terms)

  • Hardware — The physical devices you can touch: computers, tablets, printers, projectors, routers (boxes that connect to the internet).
  • Software — The programs that run on hardware. Examples: a typing program, a web browser (for visiting websites), an email app.
  • Network — A group of devices connected so they can share information (for example, the school's computers connected to the same Wi-Fi).
Daily example: Your teacher types class notes on a computer (software). The notes appear on the projector (hardware) so everyone sees them. The same notes can also be emailed to your parents (network & software).

Examples of IT in OUR school — explained

1. Computers (desktop & laptop)

Uses: learning programs, typing, saving students' work, playing educational games, showing presentations.

Tip: A single computer can be used for many tasks — that is why devices are flexible.

2. Tablets

Uses: reading e-books, drawing with a stylus, learning apps, taking photos for class projects.

3. Projector / Interactive whiteboard

Uses: display teacher slides, videos, interactive lessons where students touch or write on the board.

4. Printer / Scanner / Copier (multifunction)

Uses: print worksheets and certificates, make copies of test papers, scan students' work to a computer.

5. School Wi-Fi & Internet

Uses: allows devices to find information online, access educational websites, send emails and work with cloud documents.

6. Server and School Management Systems

Uses: store school data, save pupil records and allow teachers to access shared resources and results.

7. Cameras & Security Systems

Uses: security (CCTV) and sometimes to record school events or lessons for later viewing.

Real life: When a teacher prints an assessment, the teacher used a computer to create the assessment and a printer to print it — two IT tools working together.

How to sort school IT (by purpose)

We can group devices by what they do. This helps when answering exam questions — list the group, then give examples and short explanations.

Why one device can have many uses — explained for exam answers

Give a clear reason and an example in exam answers:

  • Reason: Many devices run different programs (software). The same hardware can run different software to perform different jobs.
  • Example answer (short): A computer can be used for typing (word processor), research (web browser), and showing slides (presentation software).

Exam tip

When asked “Explain why...” use this structure: point → reason → example. Example: “A tablet can be used for reading, drawing and recording because it has a touchscreen, camera and apps to do these jobs.”

Glossary — Important words to learn

  • Device: A machine such as a computer or tablet.
  • Hardware: Physical parts you can touch (monitor, keyboard, printer).
  • Software: Programs that tell a computer what to do (apps, browsers).
  • Network: A group of connected devices that share information.
  • Wi-Fi: Wireless connection that allows devices to access the internet without cables.
  • Server: A powerful computer that stores data for other computers to use.

Practice: Exam-style questions & model answers

Short-answer example

Q: Name two devices used for learning in school.

Model answer: Computers and tablets — both run learning apps and show information.

Longer-answer example

Q: Explain two ways the school uses IT to help teachers and pupils.

Model answer: The school uses projectors to show lesson slides and videos to the whole class, which helps pupils see large images and follow lessons. The school also uses printers to print worksheets and tests so that pupils can do practice on paper. These tools help teachers prepare lessons and provide materials for pupils to learn from.

Exam tip

Always include one example in your answer. If a question asks “explain”, give a reason and an example.
