Year 3 Computing — Week 2

Topic: Input and Output Devices | Date: Week 2

Objective: Students will learn to identify input and output devices.

Learning Objective

At the end of the lesson, students should be able to identify and classify input and output devices, describe a simple process, and design a digital device.

Success Criteria

  • ✔️ I can classify input and output devices correctly.
  • ✔️ I can describe a simple input–process–output system.
  • ✔️ I can design a basic digital device that uses input and output.

Notes and Explanations

What are input devices?
Input devices are tools we use to give instructions or data to a computer. They allow us to “put information in.” Examples include: keyboard, mouse, microphone, scanner, and touchscreen.

What are output devices?
Output devices are tools the computer uses to send information back to us. They allow us to “get information out.” Examples include: monitor, printer, speakers, and headphones.

The Input–Process–Output Model:
- Input: You press keys on a keyboard.
- Process: The computer understands the letters.
- Output: The letters appear on the screen.

Real-life examples:

  • In a cinema, a projector is an output device showing a movie, while the remote control is an input device.
  • A bank ATM uses a keypad (input), processes your request, then gives you money or shows your balance on screen (output).
  • A digital camera takes pictures (input), processes them, and then displays them on the screen (output).

Key Concepts

  • Every digital system has input, process, and output.
  • Some devices (like touchscreens) can be both input and output.
  • Understanding input and output helps us design new technology.
