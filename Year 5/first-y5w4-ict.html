Key Notes and Definitions

  • Search Engine: A website that helps people find information online, such as Google, Bing, or Yahoo.
  • Web Crawler: A program used by search engines to scan and collect information from websites automatically.
  • Index: A huge list of all the pages that a search engine knows about, like a library catalogue.
  • Search Term: The word or phrase you type into a search bar to find information.
  • Ranking: The order that search results appear in — the most relevant results show up first.

How Search Engines Work

Search engines use special software called web crawlers to visit websites and gather data. This data is stored in a massive index. When you type a search term, the search engine looks through its index — not the whole internet — to find the most relevant pages. It then ranks them so the best matches appear first.

For example, if you search for “solar system facts”, the engine will look for pages that contain those words, are trustworthy, and are most often visited by others.

Why Do We Need Search Engines?

  • The internet has billions of websites — finding one manually would take forever!
  • Search engines organise this information for quick access.
  • They use algorithms to show results that best match your search words.

Common Misconceptions

  • ❌ Search engines search the whole internet directly. — In truth, they search their own index built by web crawlers.
  • ❌ All search results are equally reliable. — Some may be false or biased, so we must check sources.
  • ❌ Web crawlers are humans. — They are automated computer programs.

Examples and Activities

  • Try searching "fastest land animal" — How does the search engine choose which site to show first?
